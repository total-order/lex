# Installation

Can be managed using
[yarn](https://yarnpkg.com/en/docs),
[npm](https://docs.npmjs.com),
or [jspm](https://jspm.org/docs).


### yarn
```terminal
yarn add @total-order/lex
```

### npm
```terminal
npm install @total-order/lex --save
```

### jspm
```terminal
jspm install npm:@total-order/lex
```
